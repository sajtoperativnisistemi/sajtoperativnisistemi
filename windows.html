Šta je Windows?

Windows je operativni sistem razvijen od strane Microsoft-a. Pojavio se 1985. godine i postao je jedan od najpopularnijih operativnih sistema. Windows je poznat po širokoj podršci za aplikacije, korisničkom interfejsu sa grafičkim elementima i širokom rasprostranjenosti u svetu korisnika računara.

Ako želite da znate da koja je verzija bila aktivna u određenoj godini kliknite na dugme.

Windows 95

Windows 95 je bio revolucionaran korak napred za Microsoft i računarsku industriju. Ovo je prva verzija Windows-a koja je u potpunosti koristila grafički korisnički interfejs (GUI) koji je omogućavao korisnicima lakše upravljanje sistemom.

Windows XP

Windows XP je postao jedan od najpopularnijih i najdugotrajnijih Windows operativnih sistema. On je kombinovao stabilnost i pouzdanost sistema Windows NT/2000 sa poboljšanim korisničkim interfejsom namenjenim široj publici.

Windows 10

Windows 10 je donio mnoge inovacije u Windows ekosistem, uključujući unifikaciju korisničkog iskustva na različitim uređajima, kontinuirane ažuriranja, Cortana, Windows Hello, DirectX 12 i mnoge druge.
